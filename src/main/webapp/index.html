We are developing an application that allows vouchers management for a company that wants a better work organization for each employee.

The application is structured on several modules:

  1. Tickets

    • In this module you can create tickets sorted by project type, the tickets can be assigned to a specific employee and can be moved thru some statuses


  2. Contracts

    • In this module you (as admin) can create contract for you employees and for your clients , contracts which will help you to create accounts.


  3. Accounts

    • You can create employee account . and see details about those accounts


  4. Invoices

    • In this module, create and send to customers' bills when a ticket is made and accepted by it.
